Conway's Game of Life in JavaScript

This is my JavaScript implementation of Conway's "Game of Life". It currently features the original version of the game exclusively, but more variations will be added if there is time.
Please note, that the game grid is configured "Torus-like". This means every cell movement outrunning borders will come out on the other side. Again, a configuration for that will be added if there is time.

If you want to, you can fork me on GitHub.
